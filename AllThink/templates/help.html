    II. How to use main functions of ALLTHINK?

    1. Manage account:

    It includes two tags : Profile an File Manager.

    * Profile allows user provides theirs own information and upload avatar:

    - To upload avatar user just need follow steps:
    - Click button Browse.
    - Choose an image in computer.
    _ Click Save to finish.

    * File manager allows user can upload documents:

    - Click the button Browse.
    - Chose file to upload.
    - Click upload to finish.
    - Click button x to delete uploaded file.

    2. lesson manager:

    It includes two tags: Your lesson and lesson joined.

    * Your lesson allows user views all their lesson and add new lesson.
